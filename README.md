# Criminal Intelligence Q&A System
> - Neo4j for storage.
> 
> - Flask, cytoscape.js for displaying sub graph.
> 
> - Pytorch Stacked BiLSTM model each with one layer for query classifying and analysing.
> 
> - Authors: Da Chen, Drogo Zhang.
